# 班委管理操作指南

## 概述

本指南介绍如何在若依管理系统中有效管理班委用户，包括查看、分配和管理班委角色的完整流程。

## 系统架构说明

### 部门结构
- **西安文理学院** (根部门)
  - **信息工程学院** (学院部门)
    - 软件工程、计算机科学、人工智能等专业部门
  - **机械与材料工程学院** (学院部门)
  - **学生部门** (功能部门)
  - **辅导员部门** (功能部门)
  - **班委部门** (功能部门) - 专门用于班委用户管理

### 角色设计
- **学生角色**: 普通学生权限
- **班委角色**: 班委特殊权限
- **用户可以同时拥有多个角色**

## 班委管理操作流程

### 1. 查看所有班委用户

#### 方法一：通过角色管理（推荐）
1. 登录系统管理后台
2. 导航至：**系统管理** → **角色管理**
3. 在角色列表中找到"班委"角色
4. 点击该角色行的"更多"按钮
5. 选择"查看班委"或"分配用户"
6. 系统将显示所有拥有班委角色的用户列表

#### 方法二：直接访问角色用户页面
1. 在角色管理页面找到班委角色（角色ID: 101）
2. 点击"分配用户"
3. 查看"已分配用户"列表

### 2. 班委用户信息查看

在班委用户列表中，您可以看到：
- **用户名称**: 用户的登录名
- **用户昵称**: 用户的显示名称
- **所属部门**: 用户归属的部门（通常为班委部门）
- **邮箱**: 联系邮箱
- **手机**: 联系电话
- **状态**: 用户账号状态
- **创建时间**: 账号创建时间

### 3. 添加新班委

1. 在角色用户管理页面点击"添加用户"
2. 在弹出的用户选择框中：
   - 可以按用户名、手机号搜索
   - 选择需要添加班委角色的用户
3. 确认添加

### 4. 取消班委权限

1. 在班委用户列表中找到目标用户
2. 点击该用户行的"取消授权"按钮
3. 确认操作

### 5. 批量管理班委

1. 使用列表左侧的复选框选择多个用户
2. 点击"批量取消授权"进行批量操作

## 部门与角色的关系说明

### 重要概念
- **用户只能属于一个部门**：每个用户的`dept_id`字段只能指向一个部门
- **用户可以拥有多个角色**：通过`sys_user_role`关联表实现
- **班委用户建议归属班委部门**：便于统一管理和权限控制

### 典型配置示例
```
用户：胡淇
- 部门：班委部门 (dept_id: 202)
- 角色：学生 + 班委 (双重角色)
- 权限：拥有学生基础权限 + 班委特殊权限
```

## 查询和筛选技巧

### 1. 按角色查询用户
- 使用角色管理功能可以精确查询特定角色的所有用户
- 支持按用户名、手机号进行二次筛选

### 2. 按部门查询用户
- 在用户管理页面可以按部门筛选
- 班委部门下的用户通常都是班委

### 3. 组合查询
- 可以结合部门和角色信息进行精确定位
- 利用搜索功能快速找到特定用户

## 最佳实践建议

### 1. 班委用户管理
- **统一部门归属**：建议所有班委用户都归属于"班委部门"
- **双重角色配置**：班委用户同时拥有"学生"和"班委"角色
- **定期权限审查**：定期检查班委权限分配是否合理

### 2. 权限控制
- 根据班委职责分配相应的菜单权限
- 注意数据权限的配置，确保班委只能访问授权范围内的数据

### 3. 用户体验优化
- 利用角色管理功能进行班委的集中管理
- 使用搜索和筛选功能提高管理效率

## 常见问题解答

### Q: 为什么在学生部门看不到班委用户？
A: 因为班委用户归属于"班委部门"而非"学生部门"。系统的部门查询逻辑基于用户的`dept_id`字段，用户只能属于一个部门。

### Q: 如何查看某个专业的班委？
A: 通过角色管理查看所有班委用户，然后根据用户信息（如姓名、联系方式）或其他标识来区分专业。

### Q: 班委用户可以同时是学生吗？
A: 可以。用户可以同时拥有"学生"和"班委"两个角色，这样既有学生的基础权限，又有班委的特殊权限。

### Q: 如何批量管理班委？
A: 在角色用户管理页面使用批量选择功能，可以进行批量取消授权等操作。
