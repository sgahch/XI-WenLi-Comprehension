# 学生综合测评系统 v2.0.0 升级使用手册

## 版本信息

- **版本号**：v2.0.0
- **发布日期**：2025-11-09
- **升级类型**：重大功能升级
- **适用对象**：系统管理员、辅导员、学生

---

## 一、升级概述

### 1.1 升级内容

本次升级主要包括以下重大改进：

1. ✅ **学号登录系统**
   - 学生使用学号登录（如：2507240101）
   - 教职工使用工号登录（如：T20240001）
   - 管理员使用自定义用户名登录

2. ✅ **三级部门结构**
   - 学院 → 专业 → 班级
   - 数据结构化管理
   - 自动填充学院、专业、班级信息

3. ✅ **批量导入增强**
   - 支持学生批量导入
   - 支持教职工批量导入
   - 自动创建学生档案

4. ✅ **学生档案管理优化**
   - 部门树选择器
   - 自动填充学院、专业、班级
   - 数据一致性保障

5. ✅ **旧功能废弃**
   - 隐藏"考评加分记录"菜单（已被综测填报系统替代）

## 二、用户登录指南

### 2.1 学生登录

**登录入口**：系统登录页面

**登录凭证**：
- 用户名：学号（10位数字，如：2507240101）
- 初始密码：123456

**登录步骤**：
1. 打开系统登录页面
2. 在"账号"输入框输入学号
3. 在"密码"输入框输入密码
4. 输入验证码
5. 点击"登录"

**首次登录提示**：
- 系统会提示"学生请输入学号，教职工请输入工号"
- 首次登录后请立即修改密码

**学号格式说明**：
```
2507240101
│││││└─ 序号（01-99）
││││└── 班级（01-99）
│││└─── 专业（01-99）
││└──── 年份后两位（25=2025年）
└───── 年份前两位（25=2025年）
```

---

### 2.2 教职工登录

**登录入口**：系统登录页面

**登录凭证**：
- 用户名：工号（如：T20240001）
- 初始密码：123456

**登录步骤**：
1. 打开系统登录页面
2. 在"账号"输入框输入工号
3. 在"密码"输入框输入密码
4. 输入验证码
5. 点击"登录"

**工号格式说明**：
```
T20240001
│└─ 年份+序号
└── 类型标识（T=Teacher教师，F=Faculty教职工）
```

---

### 2.3 管理员登录

**登录凭证**：
- 用户名：admin（或其他自定义用户名）
- 密码：admin123（或已修改的密码）

---

### 2.4 密码修改

**修改路径**：个人中心 → 修改密码

**步骤**：
1. 登录系统
2. 点击右上角头像 → 个人中心
3. 点击"修改密码"
4. 输入旧密码
5. 输入新密码（两次）
6. 点击"确定"

**密码要求**：
- 长度：5-20个字符
- 不能包含非法字符：< > " ' \ |

## 三、部门管理指南

### 3.1 部门结构

**三级结构**：学院 → 专业 → 班级

```
西安文理学院（100）
├── 信息工程学院（101）
│   ├── 软件工程专业（103）
│   │   ├── 软工2401班（500）
│   │   ├── 软工2402班（501）
│   │   ├── 软工2301班（502）
│   │   └── 软工2302班（503）
│   └── 计算机科学与技术专业（104）
│       ├── 计科2401班（510）
│       ├── 计科2402班（511）
│       └── 计科2301班（512）
└── 机械与材料工程学院（102）
    └── 机械工程专业（210）
        ├── 机械2401班（530）
        └── 机械2402班（531）
```

---

### 3.2 部门管理操作

**管理路径**：系统管理 → 部门管理

#### 3.2.1 新增部门

1. 点击"新增"按钮
2. 选择上级部门
3. 填写部门名称
4. 填写显示排序
5. 选择部门状态（正常/停用）
6. 点击"确定"

**注意事项**：
- 学院的上级部门为"西安文理学院"
- 专业的上级部门为对应学院
- 班级的上级部门为对应专业

#### 3.2.2 修改部门

1. 点击部门后的"修改"按钮
2. 修改部门信息
3. 点击"确定"

#### 3.2.3 删除部门

1. 点击部门后的"删除"按钮
2. 确认删除

**注意**：删除部门前请确保该部门下没有用户或子部门

## 四、用户管理指南

### 4.1 单个用户创建

**管理路径**：系统管理 → 用户管理

**步骤**：
1. 点击"新增"按钮
2. 填写用户信息：
   - 用户名称：真实姓名
   - 归属部门：选择部门（学生选班级，教职工选学院/专业）
   - 手机号码：11位手机号
   - 邮箱：电子邮箱
   - 用户名：学号或工号
   - 密码：初始密码（建议123456）
   - 用户性别：男/女/未知
   - 状态：正常/停用
   - 岗位：选择岗位
   - 角色：选择角色（学生/辅导员/班委等）
3. 点击"确定"

**学生用户特殊说明**：
- 用户名填写学号（如2507240101）
- 归属部门选择班级（如软工2401班）
- 角色选择"学生"
- 系统会自动创建学生档案

---

### 4.2 批量导入用户

**管理路径**：系统管理 → 用户管理 → 导入

**步骤**：
1. 点击"导入"按钮
2. 点击"下载模板"
3. 按照模板格式填写数据（详见《批量导入用户指南.md》）
4. 上传填写好的Excel文件
5. 选择是否"更新已存在数据"
6. 点击"确定"
7. 查看导入结果

**导入字段**：
- 部门编号（必填）
- 登录名称（必填）
- 用户名称（必填）
- 用户邮箱（选填）
- 手机号码（选填）
- 用户性别（选填）
- 账号状态（选填）

**详细说明**：请参考《批量导入用户指南.md》和《Excel模板字段说明.md》

---

### 4.3 用户信息修改

**步骤**：
1. 在用户列表中找到目标用户
2. 点击"修改"按钮
3. 修改用户信息
4. 点击"确定"

---

### 4.4 重置密码

**步骤**：
1. 在用户列表中找到目标用户
2. 点击"重置密码"按钮
3. 确认重置
4. 密码将重置为系统默认密码（123456）

---

## 五、学生档案管理指南

### 5.1 查看学生档案

**管理路径**：学生档案管理 → 学生档案管理

**功能**：
- 查看所有学生档案
- 按学号、学院、专业、班级等条件搜索
- 导出学生档案数据

---

### 5.2 新增学生档案

**步骤**：
1. 点击"新增"按钮
2. 填写学生信息：
   - 关联用户ID：选择已创建的用户
   - 学号：学生学号
   - 所属班级：选择班级（使用部门树选择器）
   - 学院名称：自动填充
   - 专业：自动填充
   - 年级：手动输入（如2024级）
   - 班级名称：自动填充
   - 性别：男/女/未知
   - 出生日期：选择日期
   - 入学日期：选择日期
   - 宿舍：宿舍号
   - 家庭地址：详细地址
   - 紧急联系人：联系人姓名
   - 紧急联系电话：联系人电话
3. 点击"确定"

**重要提示**：
- 选择"所属班级"后，学院、专业、班级名称会自动填充
- 不需要手动输入学院、专业、班级名称

---

### 5.3 修改学生档案

**步骤**：
1. 在学生档案列表中找到目标学生
2. 点击"修改"按钮
3. 修改学生信息
4. 点击"确定"

---

### 5.4 删除学生档案

**步骤**：
1. 在学生档案列表中找到目标学生
2. 点击"删除"按钮
3. 确认删除

**注意**：删除学生档案不会删除用户账号

## 六、常见问题FAQ

### Q1: 学生登录时提示"用户不存在"？
**A**: 请检查：
1. 学号是否输入正确
2. 该学生账号是否已创建
3. 联系管理员确认账号状态

### Q2: 批量导入时提示"部门编号不存在"？
**A**: 请检查：
1. Excel中的部门编号是否正确
2. 该部门是否在系统中已创建
3. 参考《批量导入用户指南.md》中的部门ID对照表

### Q3: 学生导入成功，但没有创建学生档案？
**A**: 请检查：
1. 导入的用户是否分配了学生角色（角色ID=100）
2. 部门编号是否为班级ID（500-999）
3. 查看系统日志确认是否有错误

### Q4: 如何批量修改学生信息？
**A**: 重新准备Excel文件，勾选"更新已存在数据"，再次导入即可

### Q5: 忘记密码怎么办？
**A**: 
- 学生/教职工：联系管理员重置密码
- 管理员：联系系统技术支持

### Q6: 如何区分学生和教职工账号？
**A**: 
- 学生：用户名为学号（纯数字）
- 教职工：用户名为工号（字母开头）
- 角色不同：学生角色、辅导员角色等
